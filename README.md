# 수빈은행 (Subin Bank) Frontend

수빈은행 서비스의 **프론트엔드 웹 애플리케이션**입니다.  
백엔드 API 서버와 연동하여 회원가입, 로그인 등 사용자 인증 흐름을 제공합니다.

## Tech Stack
- React
- React Router DOM
- Axios
- JavaScript (ES6+)
- CSS (or styled-components / Tailwind 등 사용 시 수정)

## Features
- 회원가입 / 로그인 UI
- 인증 상태 기반 페이지 라우팅
- 백엔드 API 연동
- 기본적인 폼 검증 및 에러 처리
